# In-Game Advertisement Guidelines  

### Available Sizes  

Clay.io provides 3 sizes for advertisements. Available sizes are:  

- 320 x 50 (banner)
- 320 x 480 (interstitial)
- 480 x 320 (interstitial, landscape)

### Placement

Interstitials will always overlay the entire view.
